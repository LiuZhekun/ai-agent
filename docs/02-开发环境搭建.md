# 开发环境搭建

## 目的

提供统一的本地开发环境准备步骤，避免版本差异导致运行失败。

## 前置条件

- Windows / Linux / macOS 任一开发机
- 具备管理员权限（安装运行时）

## 依赖版本

- JDK 17+
- Maven 3.9+
- Node.js 18+
- Docker 24+（推荐）
- MySQL 8.x、Redis 7.x（可由 Docker 提供）

## 步骤

1. 安装 JDK，并设置 `JAVA_HOME`。
2. 安装 Maven，确认 `mvn -v` 正常。
3. 安装 Node.js，确认 `npm -v` 正常。
4. 启动基础中间件（推荐 `deploy/install-all.sh`）。
5. 导入数据库初始化脚本（若未自动导入）。

## 示例命令

```bash
java -version
mvn -v
node -v
cd deploy && ./install-all.sh
```

## 常见问题

- Maven 下载慢：配置国内镜像。
- Windows 执行 `.sh` 失败：建议 WSL/Git Bash 或手动执行对应 Docker 命令。
