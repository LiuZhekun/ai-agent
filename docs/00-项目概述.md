# 项目概述

## 目的

说明 AI Agent 项目的目标、边界与模块职责，帮助新成员快速建立全局认知。

## 前置条件

- 已阅读仓库根 `README.md`
- 具备 Java/Spring Boot 与基础前端经验

## 项目目标

1. 提供可扩展的 Agent 引擎（会话、工具、记忆、澄清、规划）。
2. 提供分层知识能力（L0/L1/L2/L3 RAG）。
3. 提供统一 Starter，降低业务接入成本。
4. 提供可运行 Demo 与前端 UI，支持端到端验证。

## 模块说明

- `ai-agent-core`：核心运行时。
- `ai-agent-knowledge`：知识增强与 SQL 安全。
- `ai-agent-vectorizer`：向量化与文档解析。
- `ai-agent-translator`：字段翻译与冲突策略。
- `ai-agent-spring-boot-starter`：自动装配层。
- `ai-agent-demo`：示例业务项目。
- `ai-agent-ui`：Vue 对话控制台。
- `deploy`：一键部署与离线安装。

## 示例命令

```bash
mvn clean package -DskipTests
```

## 常见问题

- 看不懂模块关系：先读 `01-架构设计.md`。
- 不知道先跑什么：按 `02 -> 03 -> 04` 的顺序执行。
