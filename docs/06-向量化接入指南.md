# 向量化接入指南

## 目的

指导业务数据与文档接入向量库，支撑语义召回和知识增强。

## 前置条件

- Milvus 服务可用
- 已引入 `ai-agent-vectorizer`

## 接入步骤

1. 在实体上标注 `@VectorIndexed`。
2. 配置同步参数（批量、定时、开关）。
3. 调用全量同步接口做首次构建。
4. 启用增量定时任务持续更新索引。

## 示例命令

```bash
curl -X POST http://127.0.0.1:8080/api/agent/vector/sync
```

## 文档解析说明

- PDF：按页/段提取文本。
- Word：读取段落文本。
- Excel：按行转换为“列名:值”。

## 常见问题

- 同步慢：调小批次并观察向量写入吞吐。
- 向量检索命中低：优化文本模板，补充业务关键词。
